# Robtaylor Claude-tools

## How do I install these formulae?

`brew install robtaylor/claude-tools/<formula>`

Or `brew tap robtaylor/claude-tools` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "robtaylor/claude-tools"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
