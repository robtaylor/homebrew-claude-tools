# robtaylor-claude-tools

Formulae available:
 * `claude-tools`; A suite of Unix-style utilities for managing Claude Code conversations, implementing a Claude Virtual Filesystem (CVF) abstraction.

## How do I install these formulae?

`brew install robtaylor/homebrew-claude-tools/<formula>`

Or `brew tap robtaylor/homebrew-claude-tools` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "robtaylor/claude-tools"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
